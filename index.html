INSTRUCTIONS
- You will be shown a square canvas
- The first time you click the canvasyou will select a start node (green square)
- The second time you click the canvas you will select an end node (orange sqaure)
- After the above have been selected you can click, hold and drag on the canvas to create walls (black squares)
- After all of the above you can select an algorithm to render onto the canvas with the selected nodes
This web application is functional only for PC users
